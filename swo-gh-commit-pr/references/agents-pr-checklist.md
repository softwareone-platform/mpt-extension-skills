# AGENTS Commit + PR Checklist

Use this checklist after reading repository `AGENTS.md`.
If repo policy conflicts with this checklist, `AGENTS.md` wins.

## Commit

- Message format: `<type>: <subject>`
- Allowed type: `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, `chore`, `ci`
- Subject imperative, under 72 chars, no period

## PR

- Reuse existing PR branch when possible
- Keep one commit in PR branch unless instructed otherwise
- PR title: `<pr_key>: <description>`
- Extract PR key from branch name `<pr_key>-<branch_description>`
- PR description starts with:

`🤖 **AI-generated PR** — Please review carefully.`

## Review Reply Rules

- Push code updates first
- Reply inline to each open thread only about that thread's fix/status
- Every GitHub comment ends with a blank line and:

> 🤖 Generated by AI
